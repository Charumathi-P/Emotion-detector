# 😊 AI Emotion Detector

An AI-powered web application that detects human emotions from facial expressions in real time using **Python, OpenCV, TensorFlow, and Flask**.

## 🚀 Features

- 📷 Real-time webcam emotion detection
- 🖼️ Detect emotions from uploaded images
- 😊 Supports multiple emotions:
  - Happy
  - Sad
  - Angry
  - Fear
  - Surprise
  - Neutral
  - Disgust
- 📊 Displays confidence score
- 🌐 User-friendly web interface
- ⚡ Fast and accurate emotion prediction

## 🛠️ Tech Stack

- OpenCV

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/ai-emotion-detector.git
```

Go to the project folder

```bash
cd ai-emotion-detector
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000/
```
