# 😊 AI Emotion Detector

An AI-powered Emotion Detection application that predicts human emotions from facial expressions, text, or voice using Machine Learning and Deep Learning.

## Features
- Real-time webcam emotion detection
- Image upload
- Text emotion analysis
- Confidence score
- Responsive UI
- Dark mode
- Emotion history

## Tech Stack
- Python
- OpenCV
- TensorFlow
- Flask
- HTML
- CSS
- JavaScript

## Installation
git clone https://github.com/yourusername/emotion-detector.git
cd emotion-detector
pip install -r requirements.txt
python app.py
